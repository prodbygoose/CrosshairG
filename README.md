CrosshairG v1.0

A lightweight, high-performance crosshair overlay for Windows 10/11.
Designed to work seamlessly across games — including fullscreen.

-- Overview --

CrosshairG is a simple utility that lets you place a customizable crosshair at the center of your screen. It runs quietly in the background and is optimized for minimal performance impact.

Built entirely in C++ for speed, stability, and low resource usage.

-- Features --
Works in fullscreen and borderless windowed modes
Clean system tray integration
Fully customizable crosshair styles
Global toggle hotkey
Minimal CPU & memory usage
No bloat, no tracking, no background services

-- Controls: --

Left-click tray icon → Open settings
Right-click tray icon → Open menu
F1 → Toggle crosshair on/off
Customization

You can fully customize your crosshair, including:

Style (Cross, Dot, Circle, etc.)
Size
Thickness
Center gap
Outline
Color

All settings are automatically saved locally.

-- Performance --
CrosshairG is built using native C++ and Win32 APIs to ensure:

Extremely low latency
Near-zero performance impact
Smooth overlay rendering
Compatibility
Windows 10 / Windows 11
Works with most games and applications

-- Privacy --
CrosshairG does not collect or transmit any data.
Everything runs locally on your machine.
